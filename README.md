# 🤖 Telegram Bot: Фото + Голос + Перевод

Современный Telegram-бот на **aiogram 3**, который:

- сохраняет все отправленные пользователем 🖼️ фото в папку `img`
- отправляет голосовое сообщение по команде 🎤 `/voice`
- переводит любой обычный текст пользователя 🌍 на английский язык
- показывает красивое стартовое сообщение и
- удобную современную клавиатуру

---

## ✨ Возможности проекта

### 1. 🖼️ Сохранение фотографий
Любая фотография, отправленная пользователем боту, автоматически сохраняется в папку `img` внутри проекта.

### 2. 🎤 Голосовое сообщение

По команде `/voice` или по кнопке **🎤 Голос** бот создает и отправляет голосовое сообщение.

### 3. 🌍 Перевод текста на английский

Любой обычный текст, который отправляет пользователь, бот автоматически переводит на английский язык.

### 4. Удобный интерфейс

Бот показывает понятную reply-клавиатуру с кнопками:
- 🚀 Старт
- ℹ️ Помощь
- 🎤 Голос

### 5. Чистая архитектура

Проект разбит на папки:
- `handlers` — обработчики сообщений
- `services` — полезная логика проекта
- `keyboards` — клавиатуры
- `img` — папка для сохраненных фотографий

## 🚀 Запуск проекта

- Установить зависимости - ввести в Терминале: pip install -r requirements.txt
- Заполнить файл .env
Пример:
BOT_TOKEN=ваш_токен_бота
VOICE_TEXT=Привет! Я отправляю тебе голосовое сообщение. Этот бот умеет сохранять фотографии и переводить твои сообщения на английский язык.
- Запустить файл - ввести в Терминале: python main.py


## 📌 Команды бота

- 🚀 /start — приветствие
- ℹ️ /help — справка
- 🎤 /voice — голосовое сообщение
